
# Drone Optimisation

## Project Overview
This project focuses on the design and structural optimization of a quadcopter drone frame capable of withstanding high temperatures while maintaining strength and durability. The drone was analyzed using ANSYS Static Structural to evaluate its mechanical performance under applied loads and pressure conditions.

## Objectives
- Design a strong and lightweight quadcopter frame.
- Improve the drone's ability to withstand high-temperature environments.
- Analyze stress, deformation, and structural strength using ANSYS.
- Optimize the frame for enhanced durability and reliable performance.

## Software Used
- ANSYS Mechanical 2023 R1
- CAD Model (Drone Frame)

## Analysis Performed
- Static Structural Analysis
- Pressure Loading
- Stress Distribution
- Deformation Analysis

## Applications
This design is suitable for drones operating in demanding environments such as:
- Wildfire monitoring
- Industrial inspections
- Disaster response
- High-temperature environments
